Wi-Fi를 이용한 수업관리시스템 (Web) 



[ 소스코드의 용도 ] 

관리자 웹을 위한 소스코드가 담겨져 있습니다. 
다음과 같은 기능을 담고 있습니다. 

- Web 상에서 Excel 데이터를 Import 하여 DB에 저장하는 기능
- Camera 모듈, C언어로 구성된 프로그램의 실행(system, exec과 같은 실행 등의 기능 )
- Controller에 해당하는 PHP 내부 동작 프로그램 
- View에 해당하는 HTML, CSS, JavaScript 등 웹 페이지 꾸며주는 부분
- Model에 해당하는 DB에 대한 DDL, DCL, DML 언어 함수화

[ 실행으로서의 제한 ]

- 이 소스코드는 'Raspberry Pi Model B' 에서의 리눅스(Raspbian)기반에서 동작하게끔 만들어졌습니다.
- PHP 프레임워크인 Codigniter에서 동작합니다. 
- 5MB의 용량제한으로 가장 중요시되는 Model, View, Controller에 대한 소스만을 제출하였습니다.


[ 참고 ] 

- 프로젝트의 전체적인 부분은 같은 디렉토리의 "프로젝트 기술서->[2]Wi-Fi 시스템"에 나와있습니다.
